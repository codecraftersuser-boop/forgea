# Forgea

**Forgea** is a modern SaaS platform designed to bridge the gap between academic education and real-world industry demands. Built for final-year Computer Science and Software Engineering students, Forgea combines structured learning, collaborative projects, peer validation, and portfolio building into a single ecosystem — transforming students into job-ready professionals.

---

## The Problem

Most CS and Software Engineering students graduate with theoretical knowledge but lack the practical experience, portfolio, and validated skills that employers look for. Traditional job platforms don't serve learners, and learning platforms don't help students collaborate on real projects or get recognized for their work.

---

## The Solution

Forgea is a career-focused platform where students can:

- Follow a **personalized learning roadmap** tailored to their chosen career path
- Collaborate on **real-world projects** with peers
- Build a **professional portfolio** backed by peer-validated skills
- Track their **employability readiness** through a transparent reputation system

---

## Target Audience

- Final-year university students in Computer Science and Software Engineering
- Beginners and intermediate learners in technology fields

---

## Career Paths

Forgea initially focuses on three high-demand specializations:

| Path | Description |
|---|---|
| **Data Engineering / Analytics / Science** | Data pipelines, analysis, machine learning, and visualization |
| **Web Development** | Frontend, backend, and full-stack web applications |
| **Mobile Development** | iOS, Android, and cross-platform mobile applications |

Each path includes a full description, required skills, career opportunities, and a structured learning roadmap from beginner to professional level.

---

## Core Features

### Personalized Learning Roadmap
Curated course recommendations based on the selected career path. Each course card displays title, institution, instructor, price, rating, difficulty, duration, and an institution prestige score. Users can filter and sort by price, community rating, prestige, recency, and level.

### Real-World Projects Marketplace
A collaborative hub where students create or join projects, search by specialization, and build their portfolios through teamwork. Project cards show technologies, team size, open roles, progress, and ownership.

### Team Recruitment System
Project creators publish open roles (Frontend Developer, Backend Developer, Data Analyst, UI/UX Designer, etc.) and students apply to join — simulating a real hiring workflow.

### Skill Validation & Reputation System
When projects are completed, team members evaluate each other. Peer reviews generate reputation points and validate skills based on actual project participation. Profile scores factor in completed courses, certifications, finished projects, peer evaluations, and technical assessments.

### Professional Profile
A portfolio-style page with photo, career path, skills, certifications, project history, reputation score, endorsements, and portfolio links. Integrates with GitHub and LinkedIn. Visually communicates employability readiness.

### Notifications Center
Real-time alerts for project applications, acceptances, ratings, course recommendations, team invitations, and project completions.

### Dashboard
A central hub displaying learning progress, recommended courses, suggested projects, pending invitations, notifications, reputation score, roadmap status, and recent achievements.

### Gamification
Achievement badges, progress bars, XP points, skill levels, milestones, and leaderboards to keep students motivated and engaged throughout their learning journey.

## Design Principles

- **Modern and professional** — comparable to LinkedIn Learning, Coursera, Notion, and GitHub in quality
- **Clean UI** — white background with blue and purple accents, clear cards, dashboards, and progress indicators
- **Fully responsive** — optimized for both desktop and mobile
